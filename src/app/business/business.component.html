NANOODLE Time

NANOODLE Time provides a simple API to the Nano community to help provide a precise and accurate timestamp for Nano transactions; this is an essential enabler for the NanoBooks functions.

Status: MVP Released

NANOODLE Books

NANOODLE Books is NANOODLE's accountancy software integration and invoicing platform that is being developed for Nano.

Status: In Development

NANOODLE Watch

NANOODLE Watch provides users with the ability to watch 1 or more Nano accounts, and get notified as new transactions occur.

Status: Future Release
