Products


NANOODLE is planning to build a range of complimentary features into their products.

NANOODLE Nano Explorer

A lightweight Nano explorer providing a platform for NANOODLE's other products.

Released:

  • Fast Asynchronous Results
  • Node and Price Synchronisation Details
  • Configurable UTC Offset with Timezone Detection
  • Configurable Currency
  • QR Code - Clickthrough to Compatible Wallets/Services
  • knano Equivalent Value

Future:

  • Real-time Language Translations (Awaiting Angular 8)
  • Mobile UI Improvements

NANOODLE Time

NANOODLE Time provides a precise timestamp for Nano transactions; this is an essential enabler for many Nano projects.

Released:

  • RPC Callback for Transactions

Future:

  • API access

NANOODLE Books

NANOODLE Books is NANOODLE's accountancy software integration platform that is being developed for Nano

Released:

  • Account Transactions CSV Download
  • OFX Download (Beta)
  • Current and Historical Prices (BTC,ETH,USD,EUR,JPY,GBP,CHF,CAD,AUD,ZAR,MXN,SEK,CNY,BRL,VES,COP,PEN,ARS)

Future:

  • Cloud accountancy software integration

NANOODLE Invoice

NANOODLE Invoice provides the ability to generate a QR code for an account to request a specific amount of Nano.

Released:

  • Basic Nano QR Code Generation

Future:

  • Advanced Invoice Generation

NANOODLE Watch

NANOODLE Watch provides the ability to watch 1 or more Nano accounts, and get notified as new transactions occur

Released:

  • Email Notifications for Account Activity

Future:

  • Configurable Notifications

NANOODLE Live!

NANOODLE Live! provides Nano network visualisations and statistics

Released:

  • Live Transactions
  • Hourly Transactions
  • Hourly Transactions Value
  • Hourly Price
  • Transaction Size

Future:

  • Longer Term Charts
